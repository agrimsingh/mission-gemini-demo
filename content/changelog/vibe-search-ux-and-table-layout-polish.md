---
title: Vibe Search UX and table layout polish
category: improvement
createdAt: 2026-03-17
---

Vibe Search now explains what it does with a short intro card, shows clickable example prompts, displays a skeleton loading state while embeddings are computed, and offers a "Show on Map" button to jump from search results to the embedding map.

The track table received several layout fixes: columns use a fixed-width layout so headers stay aligned on scroll, the "ready" status pill is hidden (only non-ready states like embedding/failed are shown), and the Preview and Explore buttons use a compact size that no longer gets cropped in tight viewports.
